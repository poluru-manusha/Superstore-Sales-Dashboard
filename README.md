# Superstore Sales Dashboard (Excel)

## Project Overview

This project analyzes retail sales data using Microsoft Excel. The goal of the project is to identify key business insights such as sales performance, profit trends, and customer purchasing behavior. The dashboard provides a clear visual representation of important metrics to help decision-making.

## Dataset

The dataset used in this project is the Superstore dataset, which contains information about:

* Orders
* Sales
* Profit
* Customers
* Products
* Regions
* Shipping details

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning Techniques

## Data Cleaning

The following steps were performed before analysis:

* Removed duplicate records
* Checked for missing values
* Standardized date format
* Verified numeric fields such as Sales and Profit
* Organized columns for easier analysis

## Key Metrics

The dashboard highlights the following important KPIs:

* Total Sales
* Total Profit
* Total Orders
* Profit Margin
* Sales by Category
* Sales by Region


## Dashboard Features

The Excel dashboard includes:

* Interactive slicers for filtering by Region, Category, and Segment
* Pivot charts for visual analysis
* Monthly sales trend analysis
* Regional sales comparison
* Top performing products

## Insights

Some key insights from the analysis:

* Certain regions generate higher sales but lower profit margins.
* Technology category contributes significantly to overall revenue.
* Seasonal trends can be observed in monthly sales.
